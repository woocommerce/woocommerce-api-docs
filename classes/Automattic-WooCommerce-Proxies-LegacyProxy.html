LegacyProxy
in package
  • WooCommerce

src/Proxies/LegacyProxy.php : 18

Proxy class to access legacy WooCommerce functionality.

This class should be used to interact with code outside the src directory, especially functions and classes in the includes directory, unless a more specific proxy exists for the functionality at hand (e.g. ActionsProxy). Idempotent functions can be executed directly.

get_instance_of()

src/Proxies/LegacyProxy.php : 36

Gets an instance of a given legacy class.

public get_instance_of(string $class_name, mixed ...$args) : object

This must not be used to get instances of classes in the src directory.

If a given class needs a special procedure to get an instance of it, please add a private get_instance_of_(lowercased_class_name) and it will be automatically invoked. See also how objects of classes having a static instance method are retrieved, similar approaches can be used as needed to make use of existing factory methods such as e.g. 'load'.

Parameters

The name of the class to get an instance for.
